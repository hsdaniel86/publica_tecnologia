# publica_tecnologia
Foi elaborada 4 interfaces para administração da temporada, um banco de dados sql server para salvar dados e uma classe para conexão com função para conexão do banco de ados e outro metodo para carregar tabela do banco de dados nos dataGridView.
A primeira interface é um menu para escolher qual serão os metodos que o usuario deseja executar, foi inserido quatro botÕes, botão calcular, consultar, inserir, cancelar.
A segunda interface executa calculos, retorna total placar, máximo da temporada, minimo da temporada, recorde máximo, recorde minimo os valores retornam em um textbox e um botão para cancelar.
A terceira interface executa consultas, retorna maior e menor pontuação da temporada, total de jogos e apresenta no textbox e um botão para cancelar.
A terceira interface o usuario insere dados no banco de dados onde foi inserido 5 textbox para o usuario preencher os dados para incluir no banco de dados.
